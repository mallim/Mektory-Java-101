## Important keyboard shortcuts 
These shortcust work with the **Eclipse IDE**. They may require different combinations in other tools.

####Auto format code
	 CTRL + Shift + F 	(Windows/Linux)
	 CMD + Shift + F 	(Mac)
	 
####Auto import
	 CTRL + Shift + O 	(Windows/Linux)
	 CMD + Shift + O 	(Mac)
	 
####Quick comment
	 CTRL + / 	(Windows/Linux)
	 CMD + / 	(Mac)
